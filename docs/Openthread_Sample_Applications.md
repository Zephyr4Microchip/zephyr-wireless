# Building and running the Openthread sample applications

Following table lists the example applications available in zephyr repository, and it demonstrates various use cases for the Thread protocol and underlying IEEE 802.15.4.


| S.NO |                                                Application                                                 |                       Description                                    |                              Doc link                                  |
|------|------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------|------------------------------------------------------------------------|
| 1    | [shell](https://github.com/zephyrproject-rtos/zephyr/tree/main/samples/net/openthread/shell)               | Test Thread and IEEE 802.15.4 using the OpenThread shell             | [Shell application](Openthread_shell_Application.md)                   |
| 2    | [coprocessor](https://github.com/zephyrproject-rtos/zephyr/tree/main/samples/net/openthread/coprocessor)   | Build a Thread border-router using OpenThread's co-processor designs | [Coprocessor application](Openthread_Radio_CoProcessor_Application.md) |
| 3    | [coap](https://github.com/zephyrproject-rtos/zephyr/tree/main/samples/net/openthread/coap)                 | Build a Full Thread Device (FTD) CoAP server and client              | [Coap application](Openthread_Coap_Application.md)                     |
